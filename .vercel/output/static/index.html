Votre productivité, personnalisée

FlexPad est un pavé numérique programmable conçu pour les professionnels qui veulent optimiser leur workflow. Chaque touche peut être configurée selon vos besoins.

  • Profils multiples — stockage flash ESP32
  • Interface web — Web Serial & Bluetooth
