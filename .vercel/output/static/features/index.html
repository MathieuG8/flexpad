Fonctionnalités

FlexPad regroupe clavier HID, macros, profils, rétroéclairage intelligent et maintenance firmware dans un seul appareil — configurable depuis le navigateur, branché en USB ou appairé en Bluetooth.

5×4 grille + Profile
3 types de touches + macros
OTA mises à jour sans fil
Critère FlexPad Pavé classique
Macros & médias Oui, par touche Souvent limité
Profils stockés sur l’appareil Flash ESP32 Rarement
Configuration sans logiciel Web Serial / BLE Non
Écran + encodeur Intégrés Variable
Open source / évolutif Firmware & UI web Propriétaire en général
⌨️

20 touches programmables

Grille 5×4 avec touche Profile pour changer de profil à la volée. Chaque emplacement accepte une touche simple, un modificateur (Ctrl, Shift, Alt, GUI), une action média (volume, lecture, etc.) ou une macro complète avec délais entre les frappes.

📋

Profils multiples

Créez, renommez et supprimez des profils selon vos contextes (bureau, logiciel de CAO, montage vidéo…). La configuration est persistée dans la flash de l’ESP32 : branchez ou appairez l’appareil, vos réglages sont déjà là.

🌐

Interface web — Web Serial & Bluetooth

L’outil sur /config/ tourne dans le navigateur : connexion USB via Web Serial ou sans fil via Web Bluetooth. Aucun installeur — idéal pour Windows, macOS ou Linux récents.

🎨

Rétroéclairage & ambiant

Réglage manuel 0–100 % ou mode « selon l’environnement » : le capteur de luminosité (ATmega) adapte l’intensité pour rester confortable jour et nuit.

🖥️

Écran couleur

Affichage du profil actif et d’informations utiles pilotées depuis le panneau Display de l’interface, en lien avec le firmware sur ATmega (pilote ST7789).

🔊

Encodeur rotatif

Rotation pour volume +/− (HID Consumer Control), clic pour muet — reconnu sur les systèmes majeurs sans pilote supplémentaire.

☀️

Capteur de luminosité

Mesure continue de la lumière ambiante pour caler automatiquement le rétroéclairage lorsque vous activez le mode environnement.

📤

Export / import JSON

Sauvegardez toute votre configuration dans un fichier JSON, dupliquez-la sur un autre poste ou restaurez une version antérieure en quelques secondes.

🔄

Mise à jour OTA

Envoyez un binaire firmware (.bin) depuis l’onglet paramètres : barre de progression, retour d’état et redémarrage pour garder FlexPad à jour sans câble de programmation.
