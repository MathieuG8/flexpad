À propos de FlexPad

FlexPad est un projet étudiant et produit de passion : un pavé numérique programmable qui allie électronique embarquée, firmware ouvert et une interface web moderne pour la configuration.

20 touches configurables
USB + BLE connexion flexible
Web aucune appli à installer

Notre mission

Offrir aux créateurs, développeurs et professionnels un outil physique concret pour accélérer les raccourcis, macros et contrôles média — sans sacrifier le plaisir de bricoler et d’adapter le matériel à son propre flux de travail. Chaque touche peut être une touche simple, un modificateur, une action média ou une macro avec délais, le tout sauvegardé dans la mémoire flash de l’ESP32.

Ce qui nous guide

Transparence

Documentation claire sur le matériel (ESP32-S3, ATmega328P) et des parcours de configuration explicites dans l’interface web.

Accessibilité

Configuration via navigateur avec Web Serial (USB) ou Web Bluetooth, sans dépendre d’un logiciel propriétaire lourd.

Évolutivité

Profils multiples, export / import JSON et mises à jour OTA pour faire vivre le firmware et la config dans le temps.

Pile technique

FlexPad repose sur une architecture en deux microcontrôleurs : l’ESP32-S3 assure la connectivité USB et Bluetooth Low Energy (HID), tandis que l’ATmega328P prend en charge l’écran, le rétroéclairage, le capteur de luminosité et l’encodeur rotatif. L’interface de configuration est une application web statique qui communique directement avec l’appareil.

  • ESP32-S3 — USB, BLE, stockage des profils et touches
  • ATmega328P — affichage ST7789, LED, capteur ambiant, encodeur
  • Interface Astro — pages vitrine + outil /config/
  • OTA — envoi de firmware .bin avec suivi de progression
